# A&T Market

Plateforme e-commerce hybride RDC.

## Architecture

- frontend : Next.js
- backend : Express + TypeScript
- extractor : FastAPI + Playwright
- database : PostgreSQL + Prisma

## Modules

- Auth
- Products
- Cart
- Orders
- Payments
- Tracking
- Pricing
- Extractor

## Objectif MVP

Permettre :
- vente locale
- import international par lien
- paiement Mobile Money
- suivi livraison
